VkClient
========
**Инструкция по запуску:**
1.Распаковать проект.<br />
2.Открыть папку с проектом в командной строке.<br />
3.Выполнить "$ pod install".<br />
4.Открыть файл "VkClient.xcworkspace"<br />
5.Запустить проект на эмуляторе IPhone.<br />
